# Avarice: Terminal-Based Strategic RPG

Avarice is a strategic turn-based game implemented in a text-based format, designed to run natively within the terminal. The project focuses on robust game logic, fair turn-based mechanics, and modular software architecture.

## Key Features
* **Versus Player Mode**: Local multiplayer strategic combat.
* **Versus COM Mode**: Challenging AI opponents with distinct behavioral patterns.
* **Balanced Turn System**: Fair initiative and turn-ordering mechanics.
* **Diverse Hero Classes**: Multiple character types with unique attributes and scaling.
* **Battle Spell System**: Strategic utility spells to influence combat outcomes.
* **In-Game Documentation**: Detailed mechanics and guides available via the "Game Guide" menu.

## Technical Specifications
The project is built with a focus on Clean Code and fundamental software engineering principles:
* **Core Java**: Implementation of Advanced OOP (Encapsulation, Inheritance, Polymorphism), Exception Handling, and the Java Collections Framework.
* **Design Principles**: Adherence to the DRY (Don't Repeat Yourself) principle and Modular Architecture.
* **Data Persistence**: Custom file-based database system (TXT) for team configuration and CRUD operations.
* **User Experience**: Integration of ASCII text art for visual rendering and native audio support.
* **Testing**: Maven-integrated JUnit lifecycle for unit testing and logic verification.

## Prerequisites
* **Java Development Kit (JDK)**: Version 8 or higher recommended.
* **Apache Maven**: Required for dependency management and build automation.
* **OOP Knowledge**: Familiarity with Object-Oriented Programming is recommended for codebase exploration.

## Usage
Follow these steps to build and execute the project from the terminal:
1. type "mvn compile" to build the project
2. type "mvn clean package" to build the jar package
3. In order to run the project, type "java -jar (jar-name).jar"
